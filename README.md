# Sistema de Login em Java (Console)

Projeto simples em Java que implementa um sistema de login via terminal, com cadastro de usuários e autenticação básica.

## 📌 Funcionalidades
- Cadastrar usuário (nome, email e senha)
- Impedir login com email/senha inválidos
- Listar usuários cadastrados
- Impedir login duplicado
- Persistência de dados em arquivo `.txt`
- Menu interativo no console

## 🛠 Tecnologias utilizadas
- Java (JDK)
- VS Code
- Git e GitHub
- Java IO (FileWriter / FileReader)
